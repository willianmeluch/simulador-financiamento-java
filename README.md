Simulador de Financiamento (Java)

Este foi um dos meus primeiros projetos focados em lógica mais complexa. O desafio aqui não foi só fazer o código rodar, mas entender como uma simulação bancária funciona na vida real.
O projeto foi desenvolvido para prática de Programação Orientada a Objetos.

O que eu aprendi construindo isso

Lógica de Juros: como transformar fórmulas matemáticas em código funcional.

Organização (POO): No começo, meu código era uma bagunça. Com esse projeto, entendi por que separar as classes e usar o encapsulamento facilita a vida.

Tratamento de Erros: Aprendi que o usuário sempre vai digitar algo que você não espera, e o programa não pode simplesmente "morrer".

Sobre o projeto

Aplicação em Java que simula financiamentos de imóveis, realizando cálculos de juros e parcelas.

Tecnologias
- Java
- POO

Objetivo

Praticar conceitos de orientação a objetos e organização de código.
